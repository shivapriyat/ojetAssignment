# ojetAssignment
OJET 10.1 assignment
#### Install NODE version > 10
#### Install OJET
```
npm install -g @oracle/ojet-cli@~10.1.0 
```
#### Build Code to generate web folder and serve the web contents
```
cd assignment-app
ojet build
ojet serve
```
#### View the site on browser using URL: http://localhost:8000
